## M2 智能推荐引擎

| 推荐类型 | 算法 | 冷启动 | 更新频率 |
|------|------|:---:|:---:|
| 协同过滤 | UserCF / ItemCF | 热门兜底 | 每日离线 |
| 内容推荐 | 商品向量余弦相似度 | ChromaDB 语义 | 实时 |
| 个性化推荐 | 画像加权 + 行为序列 | 新用户热门 | 实时 |
| 热门推荐 | Redis ZSet 排序 | 天然可用 | 每小时 |
| 关联推荐 | 买了又买 / 看了又看 | 热门兜底 | 每日 |
| 卖点提炼 | LLM 提取+话术生成 | — | 商品创建时 |
| 尺码推荐 | 尺寸表学习+用户信息推理 | 需尺码表数据 | 实时 |
| 搭配推荐 | 关联规则+语义相似 | 热门搭配兜底 | 每日 |

**推荐召回→排序流程：**
```
用户请求 → 多路召回(协同过滤+语义) → 粗排(LR) → 精排(BGE-Reranker) → 重排(去重/打散) → 返回
```

## M3 智能客服系统

### 知识构建与管理

| 功能 | 优先级 | 说明 |
|------|:---:|------|
| 商品知识自动构建 | P0 | 商品详情图片 OCR+多模态提取特征，自动入知识库 |
| 聊天记录学习 | P1 | 指定金牌客服记录 → LLM 提炼话术 → 注入知识库 |
| 素材图学习 | P0 | 自动识别尺码表/材质图/尺寸图，构建素材图库 |
| 自定义知识配置 | P1 | 手动上传物流/发票等规则，Prompt 指令驱动（无需FAQ） |
| 知识库管理 | P1 | 后台 CRUD、手动修正、批量导入、自动向量化 |
| 全店知识同步 | P1 | 商品更新自动触发知识库向量重建 |

### 智能回复与话术

| 功能 | 优先级 | 说明 |
|------|:---:|------|
| RAG 知识问答 | P0 | ChromaDB + BGE-M3(BM25+向量) → RRF融合 → BGE-Reranker精排 |
| 意图识别 | P0 | 12 类意图 BERT 分类 |
| 多轮对话 | P0 | Redis 上下文管理，支持澄清、追问、意图切换 |
| 千人千面话术 | P0 | 结合用户画像 + 商品卖点 + 促销政策 |
| 话术风格定制 | P1 | 场景/角色独立配置，提供示例即可模拟风格 |
| 商品对比回复 | P1 | 多维度总结差异并生成回复 |
| 自动素材图发送 | P0 | 识别意图后自动匹配发送 |

### 客服协同

| 功能 | 优先级 | 说明 |
|------|:---:|------|
| 超级助理 | P0 | 3秒生成参考话术，降低新人上手难度 |
| 复制销冠 | P1 | 学习金牌客服 → 同类问题复现销冠话术 → 团队共享 |
| 人工辅助模式 | P0 | Agent 生成参考话术，客服点"采纳"发送 |
| 采纳学习模式 | P1 | 采纳后系统自动学习修正 |
| 全自动发送模式 | P1 | 3秒生成+发送，可配置排除场景 |
| 转人工 | P1 | 3次兜底无效 → 转人工客服 |

**RAG Pipeline：** 用户问题 → Query Rewriting → BM25+向量混合检索 → RRF融合 → BGE-Reranker → Top-K
