# 智能电商全链路平台

> 集商品管理、智能推荐、智能客服、下单转化、工单售后、数据运营于一体的全链路 AI 电商平台

---

## 一、项目定位

通过 **统一 Agent 编排层**（Master Agent + 6 子Agent）让推荐、客服、订单、营销、工单、学习六大子系统协同工作，形成数据闭环。

**三种运行模式：** 全自动发送 / 人工辅助采纳 / 学习修正进化

## 三、核心功能

| 编号 | 模块 | 核心功能 |
|:---:|------|---------|
| M1 | 商品中心 | SPU/SKU管理、三级分类树、多规格、库存同步、商品向量化、商品图OCR特征提取 |
| M2 | 智能推荐引擎 | 协同过滤、RAG语义检索、卖点提炼、尺码推荐、搭配推荐、比价分析 |
| M3 | 智能客服系统 | RAG知识问答、12类意图识别、千人千面话术、风格定制、素材图自动发送、商品对比、全自动/辅助/学习三模式 |
| M4 | 购物车/订单 | 加购结算、订单生命周期、支付对接、物流跟踪、催单催付(千人千面) |
| M5 | 工单/售后 | 工单流转、智能分配、满意度评价、紧急升级、投诉专属处理 |
| M6 | 营销活动 | 优惠券、限时秒杀、拼团、钉钉/微信/飞书消息推送 |
| M7 | 数据运营面板 | 客服统计 + 商品分析 + 用户分析 + RFM分层 + 转化漏斗 |
| — | 统一用户中心 | JWT认证、角色权限、用户画像(RFM)、浏览/购买/咨询历史 |
| — | 学习系统 | 销冠话术提炼、知识库自动更新、话术修正闭环、风格微调 |
| — | 场景策略引擎 | 场景匹配、自动模式判断、策略执行、外部工具调度 |

## 四、数据闭环

```
用户浏览 → 推荐引擎 → 商品详情 → AI客服咨询
    ↑                              │
    │                              ▼
用户画像 ← 行为分析 ← 订单完成 ← 下单支付
    │                              │
    └── 售后工单 ← 客服介入 ← 退换诉求 ←┘
```

- 客服高频问题 → 反向优化推荐和知识库
- 推荐点击率低的商品 → 触发客服主动引导
- 客服采纳/修改话术 → 学习Agent提炼优化 → 全团队共享

## 五、开发计划

| 阶段 | 周期 | 内容 |
|------|:---:|------|
| **一期：核心链路** | 3周 | 统一用户中心 + 商品中心(SPU/SKU) + 智能推荐引擎 + 购物车/订单 |
| **二期：AI赋能** | 3周 | Master Agent编排 + 智能客服(RAG+风格+素材+三模式) + 学习系统 |
| **三期：运营闭环** | 2周 | 工单/售后 + 营销活动 + 数据运营面板 + 数据闭环自动化 |

**总计：8周**
