## 一期：核心链路（商品 + 推荐 + 购物车 + 订单）

```
请帮我搭建一个智能电商全链路平台的核心骨架：

技术栈：Vue3 + Vite + Element Plus + FastAPI + SQLAlchemy + MySQL + ChromaDB + Redis

后端 (FastAPI)：
1. 用户系统：注册/登录/JWT认证/角色管理（消费者/客服/管理员）
2. 商品中心：SPU/SKU管理，三级分类树，商品 CRUD，多规格配置
3. 商品向量化：创建/更新商品时自动用 BGE-M3 生成 embedding 写入 ChromaDB
4. 智能推荐 API：协同过滤 + 语义检索 + 热门排行
5. 购物车：Redis 存储，加购/修改/删除/结算预检
6. 订单：创建订单(分布式锁防重)/订单列表/订单详情/取消/支付回调

前端 (Vue3)：
1. 商城首页：轮播推荐 + 分类导航 + 热门推荐 + 个性化推荐
2. 商品中心：商品列表(分页+筛选+排序) + 商品详情(SKU选择+图文)
3. 购物车：购物车列表 + 结算页(优惠券选择+地址)
4. 用户中心：登录注册 + 个人信息 + 我的订单

请先创建项目骨架，包含完整的目录结构和前后端的基础路由。
```

## 二期：AI 赋能（Agent + 智能客服 + 协同）

```
在上一个项目基础上添加 AI 引擎层：

后端新增：
1. 意图识别服务：BERT 模型分类 8 类意图
   - product_inquiry / product_recommend / order_query / purchase_intent
   - after_sales / price_compare / promotion_inquiry / chitchat
2. Master Agent：按意图路由到子 Agent (基于 LangGraph 编排)
3. 客服 Agent：RAG 知识问答 (ChromaDB + BM25 混合检索 + BGE-Reranker 精排)
4. 推荐 Agent：协同过滤 + 语义检索 + 用户画像加权
5. 订单 Agent：下单引导、订单查询、物流跟踪、售后引导
6. 营销 Agent：活动匹配、优惠推荐、比价分析
7. 知识库管理：后台 CRUD 知识条目、关联商品、自动向量化
8. WebSocket 实时对话端点

前端新增：
1. AI 助手页面：对话窗口(WebSocket) + 意图标签 + 推荐商品卡片
2. 客服工作台：实时对话列表 + 工单创建 + 知识库检索
3. 知识库管理页：分类管理 + 知识条目 CRUD + 向量化状态

Agent 间协同：
- 客服回答问题后可附带推荐商品
- 推荐结果包含"了解更多"按钮跳转客服
- 用户说"我要退货"时，订单Agent查订单→客服Agent创建工单
```

## 三期：运营闭环（工单 + 营销 + 数据看板）

```
在上一个项目基础上添加运营模块：

后端新增：
1. 工单系统：创建/分配/流转/评论/满意度评价
2. 营销活动：优惠券管理(满减/折扣/直减)、限时秒杀、拼团
3. 数据运营面板 API：
   - 客服统计：咨询量/响应时间/满意度/热门问题
   - 商品分析：销量排行/分类占比/转化率
   - 用户分析：新增活跃留存/RFM四象限/转化漏斗
4. 用户行为追踪：浏览/搜索/加购/购买/咨询行为记录
5. RFM 模型：每日定时计算，更新用户画像分层

前端新增：
1. 数据看板：ECharts 图表 (首页综合 + 客服 + 商品 + 用户四个子看板)
2. 工单管理：工单列表 + 详情 + 流转操作 + 评价
3. 营销中心：优惠券管理 + 活动配置 + 秒杀页面
4. 用户画像页：标签 + RFM + 行为时间线

数据闭环：
- 客服高频问题自动标记，通知运营优化知识库
- 高咨询低转化商品自动标记，建议优化详情或降价
- 推荐点击率低的商品反馈给推荐引擎降权
```
